Project Gutenberg Books

Project Gutenberg is a library of over 70,000 free eBooks that is often used for natural language processing research. Here, I have conducted a similarity analysis collected ~12,000 full texts from the Project Gutenberg library under the Science Fiction, Fantasy Fiction, Young Adult Fiction, Historical Fiction, Mystery Fiction, General Fiction, Short Stories, Humor, Western Fiction, Adventure Fiction.

SpaCy text vectorization was used in order to create vector representations of length 300 for each individual book. T-SNE dimensionality reduction was then used to reduce the dimensions of each book vector to length 2, while maintaining as much information as possible about each text. Book represenations are then plotted on on a 2-D cartesian coordinate plane to show general high level comparison between texts.
